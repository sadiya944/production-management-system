# Production Management System

## Overview
The Production Management System is a web application built using CodeIgniter, designed for managing production processes, tracking progress, and handling customer requests. This system is intended for case studies and learning purposes.

## Features
- **Dashboard**: Summary of production progress and history.
- **Customer Management**: Add and update customer data.
- **Project Management**: Handle customer requests for product quantities to be produced.
- **Production Planning**: Create production plans based on customer requests, including shift assignments and production dates.
- **Shift Management**: Manage shifts involved in the production process, including leader assignments.
- **Leader Management**: Add and update leader (staff head) data.
- **Production Process**: Track the production process for each shift, including planning, machines, and materials used.
- **Machine Management**: Manage machine data, status, and usage history.
- **Material Management**: Track material data, stock levels, and usage history.
- **Reporting**: Generate production reports, including sorting of finished goods and waste.
- **Warehouse Management**: Monitor production progress against quantity requests and total finished goods.

## Installation Instructions
1. **Requirements**:
   - XAMPP
   - PHP (version 8.0.25, not tested with other versions)

2. **Setup**:
   - Copy and extract the project folder to `xampp/htdocs`.
   - Create a database named `db_production` in phpMyAdmin.
   - Import the database schema from `db/db_production.sql`.
   - Access the application via a web browser and log in using the provided credentials.
